atribut yang tersedia pada tag input beserta fungsinya

atribut fungsi nilai
autocomplete membuat form inputan memiliki sugesti on, off
autofocus automatis fokus ke form inputan yang memiliki atribut ini -
form tag input yang berada diluar tag form bisa menjadi bagian form id dari form yang dituju
formaction digunakan untuk menambahkan action pada tag input, jadi bisa memiliki dua proses yaitu tag action dari form dan dari tag input yang diberi atribut ini, lebih mudahnya tombol yang memiliki proses berbeda. url
formenctype digunakan untuk menambahkan enctype pada tag input tujuannya untuk merubah alur form application/x-www-form-urlencoded,multipart/form-data,text/plain
formmethod digunakan untuk menambahkan method pada tag input tujuannya untuk merubah alur form POST,GET
formnovalidate digunakan untuk menambahkan novalidate tujuannya untuk merubah alur form -
formtarget digunakan untuk menambahkan target tujuannya untuk merubah alur form _self,_blank
height and width untuk merubah ukuran dari form input px,%
list digunakan untuk tag <datalist></datalist> id dari tag datalist
min and max memberi batasan terhadap isi dari form input number, range, date, datetime-local, month, time and week.
multiple untuk menginput sekaligus ditujukan untuk jenis inputan : email, and file.
pattern (regexp) melakukan validasi menggunakan bahasa regex, ditujukan untuk jenis inputan : text, search, url, tel, email, and password regex
placeholder memberi penjelasan pada form input text
required membuat form tidak boleh kosong saat dikirim -
step hanya bisa menginput nilai langkah yang telah ditentukan, misal langkah 3 maka bisa mengisi 0,3,6,9. atribut ini ditujukan untuk jenis inputan : number, range, date, datetime-local, month, time and week. number
