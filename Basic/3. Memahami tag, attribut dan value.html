Tag HTML

html memiliki sebuah kode pembuka dan penutup yang disebut sebagai tag, contoh penulisan tag

<b></b>

setiap tag html ditulis dengan kode pembuka dan kode penutup, seperti tag diatas diawalin dengan kode pembuka 

<b>

dan ditutup dengan kode 

</b>

jika tidak ada kode penutup biasanya akan terjadi error pada tampilan.

Atribut dan Value HTML

didalam tag html ada lagi yang namanya atribut yang berfungsi untuk memberitahu browser data apa yang akan ditampilkan dan digunakan juga untuk mengidentifikasi element agar bisa dibaca oleh kode lainnya seperti css dan javascript.

contohnya tag 

<a href="https://www.google.com">Google</a>

kode tag a diatas memiliki atribut href dan valuenya adalah https://www.google.com

ketika dibuka dibrowser maka yang muncul adalah sebuah tulisan google, jika tulisan tersebut diklik akan mengarahkan kesebuah alamat yaitu https://www.google.com
