HTML merupakan singkatan dari Hyper Text Markup Language

pada tahun 1990 seorang ilmuan bernama Tim Berners-Lee membuat HTML untuk memudahkan para ilmuan mengakses dokumen mereka satu sama lain.

html digunakan untuk membuat sebuah halaman website.

sejarah perkembangan HTML hingga saat ini :

versi tahun
HTML 1991
HTML 2.0 1995
HTML 3.2 1997
HTML 4.01 1999
XHTML 2000
HTML5 2014
