link digunakan untuk mengarahkan halaman kehalaman lainnya. untuk membuat link di html dengan menggunakan tag <a></a>

atribut yang tersedia pada tag <a></a>

atribut fungsi nilai
href berfungsi untuk menentukan target link yang dituju alamat situs
target berfungsi untuk jenis memproses dalam membuka link tersebut secara apa _blank : membuka link dihalaman baru _self : membuka link dihalaman yang sama (pengaturan default) _parent : membuka link dihalaman frame _top : membuka link dihalaman dengan penuh framename : membuka link sesuai dengan nama frame
title berfungsi memberi kejelasan pada link yang dituju. text

penulisan kode : 

<a href="https://www.google.com/ncr" target="_blank" title="pergi ke google">kehalaman google</a>

hasil dari kode diatas

kehalaman google
