atribut yang tersedia pada tag input beserta fungsinya

atribut fungsi nilai
value untuk mengisi form yang ada -
readonly untuk membatasi fungsi jenis form yaitu hanya untuk dibaca -
disabled untuk menonaktifkan jenis form yang ada hanya terlihat saja namun tidak terbaca -
size untuk menentukan ukuran panjang form number
maxlength menentukan batas karakter number
