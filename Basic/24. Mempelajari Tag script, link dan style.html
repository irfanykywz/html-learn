tag link

tag ini digunakan untuk menggunakan resource css dari luar halaman.

contoh penulisan kode 

<link rel="stylesheet" type="text/css" href="https://www.externalweb.com/style.css"/>

dari kode diatas saya memanggil resource css berasal dari externalweb.

tag style

tag style berfungsi untuk memasukan kode css secara internal.

contoh penulisan kode 

<style type="text/css">
 body{backgroud:black;}
</style>

tag script

tag script berfungsi untuk menaruh atau memanggil kode javascript

contoh penulisan kode javascript memanggil dari luar web 

<script type="text/javascript" src="https://www.externalweb.com/script.js"></script>

contoh penulisan kode javascript menaruh didalam 

<script>
console.log("hello world");
</script>
