tag meter

tag meter digunakan untuk membuat sebuah ukuran, tampilan dari tag meter ini seperti sebuah grafik.

penulisan kode 

<meter value="6" min="0" max="10" high="5"></meter>

hasil dari kode diatas

atribut yang tersedia pada tag meter
atribut fungsi nilai
form untuk menghubungkan tag meter menjadi bagian dari form input melalui id dari form yang diplih id form
high memberikan nilai rentang tertinggi, akan merubah warna pada tag meter number
low memberikan nilai rentang terendah number
max memberi batasan nilai maksimal number
min memberi batasan nilai minimal number
optimum memberi nilai optimal number
value nilai yang akan menjadi ukuran tag meter number

tag progress

tag progress digunakan untuk menampilkan sebuah proses yang sedang berlangsung. tag progress ini bekerja dengan mudah dengan menggunakan javascript.

penulisan kode 

<progress value="5" max="100">

hasil dari kode diatas

atribut yang ada pada tag progress

atribut fungsi nilai
value nilai proses yang selesai number
max nilai maksimal dari proses number
