untuk menampilkan gambar dihalaman web maka anda harus menggunakan tag <img/>

atribut dan fungsi pada tag img

atribut fungsi nilai
src alamat sumber gambar url
alt text alternatif jika gambar tidak muncul text
width mengatur lebar px, %
height mengatur tinggi px, %
crossorigin mengijinkan gambar diakses oleh situs lain anonymous,use-credentials
srcset kumpulan url gambar dalam ukuran tertentu dan kondisi tertentu url
