atribut tag iframe

atribut fungsi nilai
src alamat situs yang akan ditampilkan di iframe url
name memberi identitas pada iframe text
width mengatur lebar px,%
height tinggi px,%
sandbox mengaktifkan fitur pembatasan pada iframe allow-forms, allow-pointer-lock, allow-popups, allow-same-origin, allow-scripts, allow-top-navigatio
srcdoc menampilkan konten html tertentu html
