struktur html 5 tidak jauh beda dengan struktur sebelumnya, hanya saja ada kode yang dikurangkan dan ditambahkan. 

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Struktur HTML 5</title>
<!--[if lt IE 9]>
<script src="https://cdnjs.cloudflare.com/ajax/libs/html5shiv/3.7.3/html5shiv.min.js"></script>
<![endif]-->
</head>
<body>

</body>
</html>

Penjelasan kode diatas

di html 5 harus menambahkan kode html5shiv.js untuk memperbaiki tampilan di browser internet explorer dibawah versi 9. 

<!--[if lt IE 9]>
<script src="https://cdnjs.cloudflare.com/ajax/libs/html5shiv/3.7.3/html5shiv.min.js"></script>
<![endif]-->

selain itu di html 5 mendifinisikan tipe enkode charset di meta dengan menambahkan kode 

<meta charset="UTF-8">
