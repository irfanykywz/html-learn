fungsi atribut id

fungsi dari atribut id adalah untuk memberi identitas terhadap tag html,nilai id ini nantinya digunakan untuk menghubungkan tag html dengan bahasa lain misalnya css dan javascript.

penulisan nilai dari id tidak boleh sama hanya boleh satu saja.

contoh penulisan atribut id 

<!DOCTYPE html>
<html>
<head>
 <title>belajar atribut id</title>
</head>
<body>

<div id="header"></div>
<div id="content"></div>
<div id="footer"></div>

</body>
</html>

fungsi atribut class

fungsi dari atribut class sama seperti id yaitu memberikan identitas terhadap tag html, nilai dari class ini nantinya digunakan untuk menghubungkan tag html dengan bahasa lain misalnya css dan javascript.

penulisan nilai dari class boleh sama dengan boleh lebih dari satu.

contoh penulisan atribut class 

<!DOCTYPE html>
<html>
<head>
 <title>belajar atribut class</title>
</head>
<body>

<div class="warna-biru"></div>
<div class="warna-hijau"></div>
<div class="warna-hijau bentuk-kotak"></div>
<div class="warna-biru bentuk-segitiga"></div>

</body>
</html>
atribut id dan class tidak bisa diterapkan pada tag base,head,html,meta,param,script,style,title
