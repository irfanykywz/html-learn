HTML memiliki kerangka utama dalam penulisannya, dibawah ini adalah kerangka utama html. 

<!DOCTYPE html>
<html>
<head>
<title></title>
</head>
<body>
</body>
</html>

untuk menuliskan kode html lainnya ditulis diantara pembuka kode head dan body, untuk kode yang dituliskan didalam kode head biasanya adalah kode html seperti title, meta, style, link, script.

sedangkan untuk kode yang ditulis didalam kode body adalah kode html seperti div,p,nav,ul,li, dan lainnya.

penjelasan kode diatas

<!DOCTYPE html> deklarasi untuk menjelaskan tipe dokumen adalah html

<html></html> merupakan element utama dari html, semua kode html ada didalamnya

<head></head> merupakan element heading, kode yang ditaruh didalam head biasanya kode yang tidak ingin ditampilkan

<title></title> merupakan kode untuk memberi judul pada halaman html

<body></body> kode html yang akan ditampilkan dituliskan didalam kode body.
