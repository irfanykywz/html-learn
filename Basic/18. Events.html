Mouse Event

atribut fungsi nilai
onclick ketika mouse diklik javascript
ondblclick ketika mouse diklik dua kali javascript
onmousedown ketika mouse scroll kebawah javascript
onmousemove ketika mouse pindah javascript
onmouseout ketika mouse keluar halaman javascript
onmouseover ketika mouse berakhir javascript

Keyboard Event

atribut fungsi nilai
onkeydown ketika keyboard ditekan javascript
onkeyup ketika keyboard dilepas javascript
onkeypress ketika keyboard diketik javascript

Form Event

atribut fungsi nilai
onblur ketika blur javascript
onchange ketika dirubah javascript
onfocus ketika fokus javascript
onreset ketika direset javascript
onselect ketika dipilih javascript
onsubmit ketika dikirim javascript

Window Event

atribut fungsi nilai
onload ketika diload javascript
onunload ketika halaman ditutup javascript
