tag baru untuk jenis form ada 2 yaitu <datalist></datalist> dan <output></output>

datalist

tag datalist digunakan untuk membuat tag form input menjadi autocomplete, jadi seperti memberi sugesti data yang sudah ada

penulisan kode 

<form>
 <input list="musim" type="text" name="musim">
 <datalist id="musim">
  <option>Spring</option>
  <option>Summer</option>
  <option>Fall</option>
  <option>Winter</option>
 </datalist>
</form>

hasil dari kode diatas

output

tag output digunakan untuk menampilkan hasil dari sebuah inputan yang berjenis perhitungan. tag output ini bekerja menggunakan javascript jadi harus ada kode javascriptnya.

contoh penulisan kode 

<form oninput="hasil.value=parseInt(var1.value)+parseInt(var2.value)">
 <input type="number" name="var1"/ value="">+
 <input type="number" name="var2" value="1" />=
 <output name ='hasil' for ='var1 var2'></output>
</form>

hasil dari kode diatas

+=
