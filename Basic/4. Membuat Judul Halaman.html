Judul halaman akan tampil pada tab browser untuk membuatnya dengan menggunakan tag <title></title>.

contoh kode untuk membuat judul halaman : 

<!DOCTYPE html>
<html>
<head>
<title>ini adalah halaman web pertama saya</title>
</head>
<body>
</body>
</html>

dari kode diatas anda sudah bisa melihat hasilnya dibrowser maka akan merubah tulisannya sesuai isi dari tag title tersebut.

Membuat judul halaman

* catatan kalau tidak ada tag title maka yang akan menjadi judul halaman web adalah alamat halaman tersebut.
