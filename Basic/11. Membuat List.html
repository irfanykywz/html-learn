membuat daftar list menggunakan tag html memiliki beberapa cara diantaranya terurut dan tidak terurut yaitu dengan menggunakan tag <ul></ul>, ol></ol>, dan <dl></dl>

membuat daftar list menggunakan tag <ul></ul>

penulisan kode : 

<ul>
 <li>sosial</li>
 <li>karir</li>
 <li>kesehatan</li>
 <li>bisnis</li>
 <li>agam</li>
</ul>

hasil dari kode diatas

  • sosial
  • karir
  • kesehatan
  • bisnis
  • agam

membuat daftar list menggunakan tag <ol></ol>

penulisan kode : 

<ol>
 <li>sosial</li>
 <li>karir</li>
 <li>kesehatan</li>
 <li>bisnis</li>
 <li>agam</li>
</ol>

hasil dari kode diatas

  1. sosial
  2. karir
  3. kesehatan
  4. bisnis
  5. agam

merubah jenis urutan untuk merubah jenis urutan menjadi huruf, romawi harus menambahkan atribut type pada tag ol. berikut daftar nilai yang tersedia pada atribut type untuk tag ol :

nilai fungsi
1 daftar list berbentuk angka
A daftar list berbentuk alphabet huruf besar
a daftar list berbentuk alphabet huruf kecil
I daftar list berbentuk angka romawi huruf besar
i daftar list berbentuk angka romawi huruf kecil

penulisan kode dengan atribut type : 

<ol type="A">
 <li>sosial</li>
 <li>karir</li>
 <li>kesehatan</li>
 <li>bisnis</li>
 <li>agam</li>
</ol>

membuat daftar list menggunakan tag <dl></dl>

penulisan kode : 

<dl>
 <dt>sosial</dt>
 <dd>- berinteraksi dengan orang</dd>
 <dt>karir</dt>
 <dd>- pendidikan atau pencapaian akademis</dd>
</dl>

hasil dari kode diatas

sosial
- berinteraksi dengan orang
karir
- pendidikan atau pencapaian akademis
