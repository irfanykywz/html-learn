atribut yang tersedia pada tag form beserta fungsinya :

atribut fungsi nilai
accept-charset memberikan jenis charset yang akan dikirim -
action alamat untuk mengirim form. (default : halaman tersebut) url
autocomplete mengisi form secara otomatis sesuai perintah browser (nilai defaultnya adalah on) on, off
enctype menentukan format data yang akan dikirim application/x-www-form-urlencoded,multipart/form-data,text/plain
method metode http yang digunakan saat pengiriman data (defaultnya : GET) GET,POST
name name digunakan sebagai identitas form untuk keperluan DOM any
novalidate form tidak akan divaliasi. -
target mengidentifikasi jenis target yang digunakan (default : _self _self,_blank
