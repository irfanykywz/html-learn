type radio

penulisan kode : 

<label><input name='gender' type="radio" value="laki-laki">laki laki</label>
<label><input name='gender' type="radio" value="perempuan">perempuan</label>

type radio digunakan untuk memilih salah satu data dari pilihan yang ada, untuk membuat pilihan maka harus memberi nilai yang sama pada atribut name.

agar tulisan bisa diklik sebagai pilihan maka tambahkan tag label sebagai pembungkus tag inputnya.

type submit

penulisan kode : 

<input type="submit" value="kirim" />

type submit digunakan untuk membuat tombol yang berfungsi untuk memproses form kehalaman action. anda bisa merubah nama tombol tersebut dengan memberi atribut value dan diberi nilai sesuai selera anda

type reset

penulisan kode : 
<form>
 <input type="text" value="wow">
 <input type="reset" />
</form>

type reset digunakan untuk membuat tombol reset yang dimana jika tombol tersebut diklik maka akan mereset semua form yang ada seperti keadaan awal.
