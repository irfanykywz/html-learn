<meta name="viewport" content="width=device-width, initial-scale=1.0">

meta viewport berfungsi untuk menentukan ukuran layar web saat dibuka pertama kali.

meta ini berfungsi untuk membuat halaman web menjadi responsive

untuk mengetahui properti apa saja yang terdapat pada atribut content meta viewport ini bisa melihat table dibawah ini

properti penjelasan
widh properti width digunakan untuk menentukan lebar halaman bisa di isi dengan nilai device-width
height properti height digunakan untuk menentukan panjang halaman bisa di isi dengan nilai device-height
initial-scale besar halaman saat dibuka, bisa di isi dengna nilai 1 agar besar mengiktui layar pengguna
minimum-scale minimal zoom halaman, bisa di isi dengan nilai 1 agar besar mengiktui layar pengguna
maximum-scale maksimal batas zoom halaman, disarankan jangan menggunakan properti ini karena jika konten berbentuk gambar maka gambar tersebut tidak dapat di zoom jika dibuka diperangkat handphone
user-scalable user-scalable digunakan untuk menentukan boleh atau tidaknya halaman di zoom, di isi dengan nilai yes atau no
