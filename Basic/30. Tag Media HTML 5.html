audio

tag audio ini digunakan untuk memutar suara didalam halaman web

penulisan kode 

<audio controls="">
  <source src="https://www.w3schools.com/html/horse.ogg" type="audio/ogg">
  <source src="https://www.w3schools.com/html/horse.mp3" type="audio/mpeg">
</audio>

hasil dari kode diatas

atribut tag audio

atribut fungsi nilai
autoplay memulai audio otomatis autoplay
controls memunculkan tombol kontrol controls
loop memutar ulang suara loop
muted meredamkan suara muted
preload menentukan audo akan diload dengan cara auto,metadata,none
src file sumber untuk audio url

embed

tag embed digunakan untuk aplikasi external, ataupun plugin

source

tag source digunakan untuk menentukan sumber file untuk tag media (picture,video,audio)

track

tag track digunakan untuk membuat daftar track pada tag media (video,audi)

video

tag video digunakan untuk menampilkan media berbentuk video

atribut tag video

atribut fungsi nilai
autoplay memulai audio otomatis autoplay
controls memunculkan tombol kontrol controls
loop memutar ulang suara loop
muted meredamkan suara muted
preload menentukan audo akan diload dengan cara auto,metadata,none
src file sumber untuk audio url
width dan height menentukan ukuran gambar px, %
poster gambar sebelum video diputar url

picture

tag picture digunakan untuk membuat gambar yang ditampilkan lebih feksibel untuk menentukan gambar yang akan ditampilkan, tag picture bisa membuat gambar tampil berbeda disetiap layar yang ukurannya berbeda.

penulisan kode 

<picture>
 <source media="(min-width: 860px)" srcset="gambar_ukuran_860.jpg"/>
 <source media="(min-width: 460px)" srcset="gambar_ukuran_460.jpg"/> 
 <img src="gambar_ukuran_width_dibawah_460.jpg"/ alt='gambar'>
</picture>
